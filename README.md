
  # README Generator

  ## Description
  This application creates a professional README based on the answers of the prompted questions in the termianl.

  ## Installation
  Make sure you have node.js installed on your computer/laptop. Then clone the Github repository onto your device. Then make sure that you install inquierer and the file system since you need both of those to run. After that you have  everything you need in order to use the generator.

  ## Usage
  Go to where you need the README to be and follow the installation instructions. Then type node index.js and you will be prompted to answer questions in the terminal. Once you have answerd the questions a README file will be available based off of you answers.
